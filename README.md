# **ConsultAR \- Estrategia Política**

Web corporativa para consultora política estratégica con estética táctica y moderna (Dark Mode / Neon Blue).

## **Características**

* Estructura **Holy Grail** en la sección de noticias.  
* Home con **Ventanas Modales** interactivas para el equipo.  
* Diseño 100% Responsivo.  
* Estética basada en inteligencia de datos y estrategia.

## **Tecnologías utilizadas**

* HTML5  
* CSS3 (Grid y Flexbox)  
* JavaScript (Vanilla)

## **Instalación**

Solo clona el repositorio y abre index.html en tu navegador.
